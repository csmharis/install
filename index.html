This guide will walk you through how to setup Visual Studio Code for C++ Programming. You will be installing the followings:

  • MS Visual Studio Code (the source code editor)
  • C/C++ VS Code extension by Microsoft (for C/C++ IntelliSense, debugging, and code browsing). This extension is compulsory.
  • Some other VS Code extensions, such as Colonize, Rainbow Bracket, etc). These are optional.
  • MinGW (Compiling and debugging tools, i.e., gcc / g++ and gdb). Note that, it is pre-built with 32-bit version only. The pre-built compiler supports up to C++ 17.
  • WinBGIm, a simple 2D graphic library.

Download VS Code and run

  • Choose Windows System Installer 64 bit.
  • Simply accept all the default settings.
  • Launch VS Code
  • Once you are in VS Code, hit CTRL P to open an input box on top of the window, and type:
    ext install ID
    This will open a market place of different type of extensions. you need to search the following extension and install it.

Extenstions to install

  • C/C++ (Microsoft)
  • Code Runner
  • Error Lens (Alexander) (Show error at coding-time) Optional
  • Better Comments (aaron-bond) (comments with colors) Optional

Download MingGW

  • Google Drive link

Note:

  • This is not an installer, instead it is a pre-built or pre-compiled version of mingw.
  • 32-bit version.
  • Supports languages up to C++17
  • WinBGIm graphic library has already been bundled with it. (Can be used for the OOP in next semester)
  • Only for Windows 7/8/10/11.
  • Open the download zip file and extract the folder mingw (from the zip file) to c:\ drive

Note:

  • Do not change the installation destination to another drive or directory.
  • !!! IMPORTANT !!! It must be c:\mingw, because the vscode configuration I made always refers to this path.
  • Hit the Windows key on your keyboard to open the Windows Start menu.
  • Type: Environment variable
  • Choose from the Windows Start Menu: Edit the system environment variables
  • Inside the window System Properties, click on the button Environment Variables.

  • Inside the window Environment Variables, under the section System variables, double-click on the variable Path.

  • Inside the window Edit environment variable, click on the button New to add a new path.Type:
    c:\mingw\bin
  • Move the entry (c:\mingw\bin) to the top making it the first entry
  • Download sample programs:
  • Sample Programs
  • Extract the zip file inside the folder to any folder and Remmember the location of file.
  • Launch VS Code from the Start Menu
  • Lunch the VS code and open the sample code.
  • You will see play button like the below Screenshot. Click and run the program.
  • If this button show bug sign you need to click the small arrow on the right of bug and select Run code

  • Create a new folder (in Any directory/Folder of your PC) . Let call this folder, programs.
  • In VS code File-> Open the folder programs into a VS Code instance.
  • Now, Press CTRL+N on keyboard - > Select Langauge -> Select C++
  • A new file will be Created. Now Save this file as FirstProgram.cpp
  • Now, you can code and Run the program.
